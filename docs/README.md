# MyMind 项目文档

MyMind 是一款对标 XMind 的思维导图工具，采用 **Web 优先、后期 Tauri 打包** 的策略，目标支持 XMind 的全部九种结构形式。

## 产品定位

- **目标用户**：需要结构化思考、知识整理、项目规划的个人与团队
- **核心价值**：流畅的键盘编辑体验 + 九种专业结构 + 可导出的高质量输出
- **平台路径**：浏览器 Web 应用 → PWA → Tauri 桌面端
